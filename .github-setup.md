2. **Click the ⚙️ gear icon** next to "About" (top right)
3. **Add the description** from above
4. **Add topics** by typing each one and pressing Enter
5. **Click "Save changes"**

## Fix Contributor Issue

### Verify Your Email on GitHub
1. Go to: https://github.com/settings/emails
2. Make sure `amyvrm@gmail.com` is listed
3. If not listed, click "Add email address" and add it
4. **Verify the email** by clicking the link sent to your inbox
5. **Optional:** Set it as your primary email

### Set Commit Email Preference
1. Go to: https://github.com/settings/emails
2. Under "Primary email address" section
3. Optionally check: "Keep my email addresses private" (this will use a GitHub noreply email)

## After Email is Verified

GitHub will automatically:
- Show you as a contributor
- Link your commits to your profile
- Update the contributor count

This may take a few minutes after verification.
